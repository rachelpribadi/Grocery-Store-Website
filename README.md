# Grocery Store Website
A grocery store e-commerce website built using HTML, CSS, and JavaScript, featuring product listings, promotions, and interactive cart functionality.

## Features
- Product catalogue with categories (groceries, snacks, frozen food)
- Add-to-cart system with dynamic quantity stepper
- Live cart counter that updates in real time
- Promotional sections and featured deals
- Recipe section with interactive pop-ups 

## Technologies Used
- HTML
- CSS
- JavaScript
